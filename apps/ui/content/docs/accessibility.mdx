---
title: Accessibility
description: Keyboard, focus, contrast, motion and zoom requirements.
---

Every primitive and particle must retain visible focus, accessible names, native form semantics and Base UI keyboard behavior. The release gate covers axe, forced-colors, reduced motion, 200% zoom, 320/390 px layouts and focus restoration.

Touch targets are at least 44 by 44 CSS pixels except dense data-table cells with an equivalent accessible action.
