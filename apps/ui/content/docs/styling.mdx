title: Styling
description: Tailwind CSS v4, data-slot and layout-only application overrides.
---

The runtime imports `@fugue/ui/globals.css`. Primitive color, typography and state styling lives in the registry; product `className` values should primarily control layout.

Tailwind `@source` includes the web app, registry app, examples and runtime package so production builds retain every statically discoverable utility.
