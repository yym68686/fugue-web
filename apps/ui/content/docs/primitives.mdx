---
title: Primitives
description: Base UI-backed building blocks and their composition contract.
---

All interactive primitives use Base UI. Triggers compose custom elements with `render`; the Radix `asChild` pattern is not part of this system.

## Required contracts

- Every part exposes a stable `data-slot`.
- Variants use typed CVA contracts.
- Dialog, Sheet and Drawer always include a title.
- Form controls use Form plus Field or FieldGroup.
- Loading buttons compose Spinner and `disabled` rather than adding an `isLoading` prop.
