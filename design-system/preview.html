Design system / Platform UI

Fugue Platform Design System

Dense enterprise-console UI tokens and components distilled into a platform-neutral layer for sidebars, topbars, data cards, lists, forms, menus, modals, and stateful feedback.

Reference DNA

The system favors Inter, 8px radii, 32-40px controls, 1px rings, low-shadow raised surfaces, compact navigation, blue primary actions, green/red metric deltas, and solid dark overlays.

Color tokens

Primitive neutrals, semantic surfaces, and status colors.
